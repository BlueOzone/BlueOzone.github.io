и§ЈеҶіж–№жЎҲеӨ§йғҪжҳҜйҖҡз”Ёзҡ„пјҢдёӢйқўжҲ‘д»¬жқҘи®Іи§ЈдёҖдёӢдёҖиҲ¬жғ…еҶөдёӢжңүд»Җд№Ҳж–№ејҸеҺ»дҝқиҜҒзәҝзЁӢе®үе…Ёй—®йўҳгҖӮ

иҫ“еҮәз»“жһңеҰӮдёӢпјҡ

еҚідҪҝеӨ§е®¶еӨҡжү§иЎҢеҮ ж¬ЎпјҢд№ҹдёҚдјҡзңӢеҲ°и¶…еҚ–е’ҢйҮҚеӨҚеҚ–зҡ„й—®йўҳпјӣеңЁдёҠйқўзҡ„д»Јз ҒдёӯжҲ‘д»¬дҪҝз”ЁдәҶsynchronizedеҗҢжӯҘд»Јз Ғеқ—дҝ®йҘ°дәҶеҚ–зҘЁзҡ„ж“ҚдҪңпјҢдҪҝеҫ—жҜҸж¬ЎеҸӘжңүдёҖдёӘзәҝзЁӢеңЁжү§иЎҢпјҢдҝқиҜҒдәҶзҘЁеҸ·зҡ„зәҝзЁӢе®үе…ЁгҖӮ

дҝқиҜҒзәҝзЁӢе®үе…Ёж–№ејҸдәҢпјҡеҗҢжӯҘж–№жі•

еҗҢжӯҘж–№жі•пјҢеҚіжңүsynchronizedе…ій”®еӯ—дҝ®йҘ°зҡ„ж–№жі•гҖӮ
з”ұдәҺjavaзҡ„жҜҸдёӘеҜ№иұЎйғҪжңүдёҖдёӘеҶ…зҪ®й”ҒпјҢеҪ“з”ЁжӯӨе…ій”®еӯ—дҝ®йҘ°ж–№жі•ж—¶пјҢеҶ…зҪ®й”ҒдјҡдҝқжҠӨж•ҙдёӘж–№жі•гҖӮеңЁи°ғз”ЁиҜҘж–№жі•еүҚпјҢйңҖиҰҒиҺ·еҫ—еҶ…зҪ®й”ҒпјҢеҗҰеҲҷе°ұеӨ„дәҺйҳ»еЎһзҠ¶жҖҒгҖӮ
synchronizedе…ій”®еӯ—д№ҹеҸҜд»Ҙдҝ®йҘ°йқҷжҖҒж–№жі•пјҢжӯӨж—¶еҰӮжһңи°ғз”ЁиҜҘйқҷжҖҒж–№жі•пјҢе°Ҷдјҡй”ҒдҪҸж•ҙдёӘзұ»гҖӮ
synchronizedдҝ®йҘ°ж–№жі•ж—¶пјҢи·ҹеҗҢжӯҘд»Јз Ғеқ—зӣёдјјпјҢд№ҹдҪҝз”ЁеҲ°дәҶй”Ғзҡ„жңәеҲ¶пјҢеҸӘдёҚиҝҮsynchronizedдҝ®йҘ°ж–№жі•ж—¶й»ҳи®Өзҡ„й”ҒжҳҜеҪ“еүҚеҜ№иұЎгҖӮ

synchronizedеҗҢжӯҘж–№жі•зӨәдҫӢд»Јз Ғпјҡ

иҫ“еҮәз»“жһңеҰӮдёӢпјҡ

е’ұд»¬дёҠйқўиҜҙеҲ°пјҢsynchronizedдҝ®йҘ°ж–№жі•ж—¶пјҢдҪҝз”ЁеҲ°зҡ„еҗҢжӯҘзӣ‘и§ҶеҷЁзҡ„еҪ“еүҚеҜ№иұЎ(this)пјҢжүҖд»ҘиҜҙеңЁдёҠйқўзҡ„дҫӢеӯҗдёӯпјҢе’ұд»¬иҰҒдҝқиҜҒдёүдёӘзәҝзЁӢдј е…Ҙзҡ„еҜ№иұЎжҳҜдёҖиҮҙзҡ„пјҢиӢҘдёҚдёҖиҮҙзҡ„иҜқе…¶е®һд№ҹе°ұжҳҜеҗ„е№Іеҗ„зҡ„дәҶпјҢдёҚз¬ҰеҗҲе”®зҘЁиҰҒжұӮгҖӮ

дҝқиҜҒзәҝзЁӢе®үе…Ёж–№ејҸдёүпјҡLock й”ҒжңәеҲ¶

д»ҺJDK 5ејҖе§ӢпјҢJavaжҸҗдҫӣдәҶжӣҙејәеӨ§зҡ„зәҝзЁӢеҗҢжӯҘжңәеҲ¶вҖ”вҖ”вҖ”вҖ”йҖҡиҝҮжҳҫзӨәе®ҡд№үеҗҢжӯҘй”ҒеҜ№иұЎжқҘе®һзҺ°еҗҢжӯҘгҖӮеҗҢжӯҘй”ҒдҪҝз”ЁLockеҜ№иұЎжқҘе……еҪ“гҖӮй”ҒжҸҗдҫӣдәҶеҜ№е…ұдә«иө„жәҗзҡ„зӢ¬еҚ и®ҝй—®пјҢжҜҸж¬ЎеҸӘиғҪжңүдёҖдёӘзәҝзЁӢеҜ№LockеҜ№иұЎеҠ й”ҒпјҢзәҝзЁӢејҖе§Ӣи®ҝй—®е…ұдә«иө„жәҗд№ӢеүҚеә”е…ҲиҺ·еҫ—LockеҜ№иұЎгҖӮ

е…ҲжқҘд»Ӣз»ҚдёӢLockжҺҘеҸЈпјҡ
java.util.concurrent.locks.Lock жҺҘеҸЈжҳҜжҺ§еҲ¶еӨҡдёӘзәҝзЁӢеҜ№е…ұдә«иө„жәҗиҝӣиЎҢи®ҝй—®зҡ„е·Ҙе…·гҖӮиҜҘжҺҘеҸЈжҸҗдҫӣжңүиҝҷдёӨдёӘж–№жі•пјҡ
(1) void lock() иҺ·еҸ–й”Ғ
(2) void unlock() йҮҠж”ҫй”Ғ

Lock й”ҒжңәеҲ¶дҝқиҜҒзәҝзЁӢе®үе…ЁиҝҮзЁӢдёӯпјҢеӨ§дҪ“дёҠеҲҶдёәдёүеӨ§жӯҘпјҡ
(1)е®һдҫӢеҢ–ReentrantLockпјӣ
(2)и°ғз”Ёй”Ғе®ҡж–№жі•пјҲиҺ·еҸ–й”Ғпјүпјҡlock()пјӣ
(3)и°ғз”Ёи§Јй”Ғж–№жі•пјҲйҮҠж”ҫй”Ғпјүпјҡunlock()пјӣ

йӮЈReentrantLockжҳҜдёӘе•ҘзҺ©ж„Ҹе‘ўпјҡ
ReentrantLock жҳҜLockжҺҘеҸЈзҡ„дёҖдёӘе®һзҺ°зұ»пјҢе®ғжӢҘжңүдёҺsynchronizedзӣёеҗҢзҡ„е№¶еҸ‘жҖ§е’ҢеҶ…еӯҳиҜӯд№үпјӣеңЁе®һзҺ°зәҝзЁӢе®үе…Ёзҡ„жҺ§еҲ¶дёӯпјҢеёёз”Ёзҡ„е®һзҺ°зұ»е°ұжҳҜ ReentrantLockгҖӮ
е®ғеҸҜд»ҘжҳҫзӨәеҠ й”ҒпјҢйҮҠж”ҫй”ҒпјӣйҖҡиҝҮlock()е’Ңunlock()й…ҚеҗҲдҪҝз”ЁпјҢжҳҜдёҖз§ҚжүӢеҠЁй”ҒпјҢжҜ”иҫғзҒөжҙ»пјӣдҪҶжҳҜжіЁж„ҸеңЁдҪҝз”ЁиҝҷдёӘй”Ғж—¶дёҖе®ҡиҰҒжіЁж„ҸйҮҠж”ҫй”ҒпјҢеҗҰеҲҷеҸҜиғҪдјҡйҖ жҲҗжӯ»й”ҒгҖӮ

2гҖҒLockжҳҜжҳҫзӨәй”ҒпјҲжүӢеҠЁејҖеҗҜе’Ңе…ій—ӯй”ҒпјҢдёҚиҰҒеҝҳи®°е…ій—ӯй”ҒпјүпјҢsynchronizedжҳҜйҡҗејҸй”ҒпјҢеҮәдәҶдҪңз”ЁеҹҹиҮӘеҠЁйҮҠж”ҫгҖӮ
3гҖҒLockеҸӘжңүд»Јз Ғеқ—й”ҒпјҢsynchronizedжңүд»Јз Ғеқ—й”Ғе’Ңж–№жі•й”ҒгҖӮ
4гҖҒдҪҝз”ЁLockй”ҒпјҢJVMе°ҶиҠұиҙ№иҫғе°‘зҡ„ж—¶й—ҙжқҘи°ғеәҰзәҝзЁӢпјҢжҖ§иғҪжӣҙеҘҪгҖӮе№¶дё”е…·жңүжӣҙеҘҪзҡ„жү©еұ•жҖ§пјҲеӣ дёәжҸҗдҫӣдәҶжӣҙеӨҡзҡ„еӯҗзұ»пјүгҖӮ

е®һйҷ…ејҖеҸ‘иҝҮзЁӢдёӯдјҳе…ҲдҪҝз”ЁйЎәеәҸпјҡ
Lockй”ҒвҖ”вҖ”>еҗҢжӯҘд»Јз Ғеқ—пјҲе·Із»Ҹиҝӣе…ҘдәҶж–№жі•дҪ“пјҢеҲҶй…ҚдәҶзӣёеә”иө„жәҗпјүвҖ”вҖ”>еҗҢжӯҘж–№жі•пјҲеңЁж–№жі•дҪ“д№ӢеӨ–пјү

зәҝзЁӢзҡ„жӯ»й”Ғй—®йўҳ

жӯ»й”ҒпјҡдёҚеҗҢзҡ„зәҝзЁӢеҲҶеҲ«еҚ з”ЁеҜ№ж–№зҡ„йңҖиҰҒзҡ„еҗҢжӯҘиө„жәҗдёҚж”ҫејғпјҢйғҪеңЁзӯүеҫ…еҜ№ж–№ж”ҫејғиҮӘе·ұйңҖиҰҒзҡ„еҗҢжӯҘиө„жәҗпјҢе°ұеҪўжҲҗдәҶзәҝзЁӢзҡ„жӯ»й”ҒгҖӮ
еҮәзҺ°жӯ»й”ҒеҗҺпјҢдёҚдјҡеҮәзҺ°ејӮеёёпјҢдёҚдјҡеҮәзҺ°жҸҗзӨәпјҢеҸӘжҳҜжүҖжңүзҡ„зәҝзЁӢйғҪеӨ„дәҺйҳ»еЎһзҠ¶жҖҒпјҢж— жі•з»§з»ӯеӨ„зҗҶе…¶д»–дәӢжғ…гҖӮ

и§ЈеҶі/йҒҝе…Қж–№жі•пјҡ
1гҖҒдё“й—Ёзҡ„з®—жі•гҖҒеҺҹеҲҷпјӣ
2гҖҒе°ҪйҮҸеҮҸе°‘еҗҢжӯҘиө„жәҗзҡ„е®ҡд№үпјӣ
3гҖҒе°ҪйҮҸйҒҝе…ҚеөҢеҘ—еҗҢжӯҘгҖӮ

зәҝзЁӢе®үе…Ёзҡ„еҚ•дҫӢжЁЎејҸ

йҘҝжұүејҸеҺҹжң¬е°ұжҳҜзәҝзЁӢе®үе…Ёзҡ„пјӣжҮ’жұүејҸйңҖиҰҒдҪҝз”ЁеҗҢжӯҘд»Јз Ғеқ—зҡ„ж–№ејҸеҺ»дҝқиҜҒзәҝзЁӢе®үе…ЁгҖӮ

жҖ»з»“

жҖ»зҡ„жқҘиҜҙпјҢдҝқиҜҒзәҝзЁӢе®үе…ЁжңүеҗҢжӯҘд»Јз Ғеқ—гҖҒеҗҢжӯҘж–№жі•е’ҢLockй”ҒиҝҷеҮ з§Қж–№жі•пјҢе®һйҷ…ејҖеҸ‘дёӯжҺЁиҚҗдҪҝз”ЁLockй”Ғзҡ„пјҢеӣ дёәдҪҝз”ЁLockй”ҒпјҢJVMе°ҶиҠұиҙ№иҫғе°‘зҡ„ж—¶й—ҙжқҘи°ғеәҰзәҝзЁӢпјҢжҖ§иғҪжӣҙеҘҪгҖӮе№¶дё”е…·жңүжӣҙеҘҪзҡ„жү©еұ•жҖ§гҖӮжң¬зҜҮе°ұе®Ңз»“е•ҰгҖӮ
